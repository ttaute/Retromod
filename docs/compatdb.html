🔬 Static audit, not gameplay. The top 5,000 Modrinth mods were run through Retromod 1.1.0-snapshot.2's transform (target MC 26.1.2, May 30 2026) and checked for unresolved Minecraft/Fabric-API references — Clean = transforms with zero unresolved references, which is necessary but not sufficient for "works in game" (runtime behavior, datapacks, and bridge-backed APIs aren't measured; bridge fixes since the audit aren't reflected). Treat Gaps as "needs Retromod work or a port", not as a verdict on the mod. Community reports above are the ground truth — add yours.

Add your report

Fill this in and hit submit — it opens a pre-filled GitHub report (you just press one button there). A bot then turns it into a pull request against the database automatically; once a maintainer merges, your report is live. No YAML, no manual issue writing. (A GitHub account is the only requirement — it's what keeps spam out.)

Power users: you can also PR docs/_data/compatdb.yml directly.
